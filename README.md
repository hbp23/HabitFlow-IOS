# HabitFlow
HabitFlow is a clean, intuitive iPadOS habit-tracking application designed to help users build and maintain healthy daily routines. 

## Features
* **Daily Tracking ("Today" View):** Easily view and check off habits scheduled for the current day.
* **Flexible Scheduling:** Create habits that repeat every day or on specific days of the week (e.g., Mon-Fri).
* **Habit Management:** Add new habits, view all active habits, and archive old ones to keep your dashboard clean.
* **Progress Statistics:** Track your success with an overview of your total/active habits, your overall completion percentage for the last 7 days, and your most frequently completed habits.
* **User Authentication:** Secure sign-in functionality to keep user data private and synced.

## Tech Stack
* **Framework:** SwiftUI
* **Architecture:** MVVM
* **Local Storage / Database:** Firebase
* **Authentication:** Firebase Auth

## Getting Started
To run this project locally:
1. Clone the repository: `git clone https://github.com/hbp23/HabitFlow-IOS.git`
2. Open `HabitFlow.xcodeproj` in Xcode.
3. Add your own GoogleService-Info.plist for Firebase.
4. Select an iPhone or iPad simulator and click **Run** (Cmd + R).
